# Network Automation Core
The central logic engine for processing network data.

## Contents
- `scripts/`: Python tools for syncing Git data to NetBox.
- `templates/`: Jinja2 "Golden Config" blueprints for Cisco hardware.
- `playbooks/`: Ansible workflows for deployment and validation.

## Role Naming Convention
We follow a 3-character role naming convention:
- `acc`: Access
- `dis`: Distribution
- `cor`: Core
- `tra`: Transit
